### Featured Project

🛡️ **[Aegis](https://github.com/Krish-Korat/Aegis)** – *Reverse Proxy Web Application Firewall (WAF)*  
Aegis is an ongoing Web Application Firewall project aimed at analyzing incoming traffic, detecting malicious payloads, and mitigating web-based attacks (such as SQLi and XSS) before they reach the backend server.  
- **Focus Areas**: Packet Inspection, Threat Detection, Traffic Filtering.
